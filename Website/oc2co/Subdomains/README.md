Curated copies of the oc2co website's subdomain/page folders:
- arcade/
- store/
- checkout/
- success/
- cancel/

Each contains the page-specific index.html .

These were copied (cp -r) from Source/oc2co_website/<name>/ during curation so the website pages have a dedicated organized view here. The originals remain inside Source/ to preserve the full build source intact (references, dev serving, etc.).

See Source/README.md and root README.md for details.
